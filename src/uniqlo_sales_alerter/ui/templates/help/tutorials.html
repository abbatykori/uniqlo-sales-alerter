Tutorials

Learn by following a sequence.

Add your first filter

  1. Open /ui/filters/new.
  2. Name it something memorable: "Me tops" or "Kid 5y" work well.
  3. Set gender + sizes. Tip: paste an invoice to auto-detect.
  4. Set min discount (e.g. 40%) and availability ("both" is fine to start).
  5. Save. The next scheduled check will use your filter.

Wire up notifications

Pick one or more Apprise URLs from ~80 supported services. Paste them into config.yaml::notifications.apprise_urls or set NOTIFICATIONS_APPRISE_URLS.
